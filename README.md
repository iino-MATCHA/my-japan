# MATCHA | My Japan

Track the Japanese prefectures you've visited on an interactive map and record your journey across Japan. A travel companion for inbound visitors, by MATCHA.

Built with React, Vite and Tailwind CSS.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```
   npm install
   ```
2. Start the dev server:
   ```
   npm run dev
   ```
   The app runs at http://localhost:3000

## Build

```
npm run build
```

The production build is output to `dist/`.

## Deploy

This is a static Vite app and deploys to [Vercel](https://vercel.com) with no extra configuration — Vercel auto-detects the Vite framework (build command `npm run build`, output directory `dist`).
